App Functionality

This app is a basic addition, subtraction and mutiplication application.
There is four main pages.

-Easy Page

This is very basic addition and subtraction two numbers only, the biggest the
numbers can be is 24.

-Regular Page

This is a bit harder adding multiplication . if multiplication is the symbol the 
biggest the numbers can be is 10. If its a plus or minus the numbers can go 
up to 48.

-Hard Page

This is the hardest level adding an extra number and symbol. The max a 
number after a multiplication symbol is max 10 and otherwise the numbers
can go up much higher.

- High Score Page

This is where your high score is kept track of in local settings for each individual
game page.
